Process

How It Works

01

Input SMILES String

Provide the molecular SMILES (Simplified Molecular Input Line Entry System) representation — the universal chemical language used in drug databases worldwide.

02

Feature Extraction via RDKit

Extracts key physicochemical descriptors: molecular weight, H-bond donors & acceptors, lipophilicity (LogP), topological polar surface area, and rotatable bond count.

03

Lipinski Rule Validation

Applies Lipinski's Rule of Five as a hard filter — automatically flagging molecules that violate oral bioavailability criteria before ML evaluation.

04

ML Score & QED Prediction

The trained classifier produces a QED score (0–1) and drug-likeness probability with confidence, giving researchers an interpretable, actionable result.
